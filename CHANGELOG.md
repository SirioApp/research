# Changelog

## 0.2.0
- Renamed package branding to Backed Research Agent.
- Added dashboard-first output model.
- Added expanded risk register with owner/timeframe fields.
- Added team, market, and fundraising assessment modules.
- Added ingestion resilience for rate-limited sources (429 retry/backoff/fallback).
- Added authenticated HTTP API (`/v1/health`, `/v1/analyze`) with API-key support.
- Added integration documentation for API-to-dashboard pipelines.
- Added API helper tests and improved release checklist for public repository readiness.
- Added public-repo documentation set and CI workflow.

## 0.1.0
- Initial release with multi-source ingestion and baseline scoring.
